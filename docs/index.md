# Documentation Index

This folder contains the consolidated documentation for the Secure Santa project. Use this index to navigate the docs.

- `setup.md` — Installation and PostgreSQL setup (quickstart + DB)

- `troubleshooting.md` — Common issues and fixes
- `update_summary.md` — Summary of recent changes
- `backend.md` — Backend-specific developer notes
- `frontend.md` — Frontend-specific developer notes

If you're using the tool for the first time, start with `overview.md`.
